# OpenAITrade Public Skill

这是面向中文用户的公共 skill 入口。

它强调的三件事是：

- 免费数据
- 免费策略

这里的免费工作流特指：

- skill 可以帮助 agent 获取免费数据
- skill 可以帮助 agent 调用免费策略
- skill 可以帮助 agent 执行回测
- skill 可以帮助 agent 做策略优化
