# Docs Guide

This folder contains the public documentation set for OpenAITrade Public.

## Recommended reading order

1. `STRATEGIES.md`
2. `data_sources.md`
3. `BACKTESTING.md`
4. `LIVE_TRADING.md`
5. `broker_adapters.md`
6. `strategy_pack_standard.md`
7. `strategy_pack_template.md`

## Notes

- `STRATEGIES.md` focuses on the free strategy library.
- `data_sources.md` and `DATA_SOURCES_FULL.md` focus on free or low-barrier data access.
- `BACKTESTING.md` covers the backtest assumptions used by the public workflow.
- `LIVE_TRADING.md` focuses on integration boundaries and risk controls.
